
To compile with the Visual C++ project:

- Install libsndfile: http://www.mega-nerd.com/libsndfile/
Compile a dll named libsndfile.dll and put the corresponding libsndfile.lib and the sndfile.h files along with the LibAudioStream.dsp file.

- Install portaudio V18 : http://www.portaudio.com/
Compile a dll (or static library) and put the corresponding PortAudio.lib and the portaudio.h files along with the LibAudioStream.dsp file.

Install PthreadWin32 library: http://sources.redhat.com/pthreads-win32/
and put the pthreadVC.lib, pthread.h and sched.h files along with the LibAudioStream.dsp file.
