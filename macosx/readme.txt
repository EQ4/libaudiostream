
To compile with the Xcode project:

- Install libsndfile: http://www.mega-nerd.com/libsndfile/
using DarwinPorts: http://darwinports.opendarwin.org/

- Install portaudio V18 : http://www.portaudio.com/
The Xcode project is expecting a "PortAudioLib" static library located along with the LibAudioStream.xcode file.

- Install jack: http://www.jackosx.com
